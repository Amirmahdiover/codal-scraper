# Fiverr Portfolio Draft

## Portfolio Project Title

CODAL Financial Reports Scraper

## Short Description

A Python web scraping and data pipeline project that collects public financial announcements, extracts income statement data, normalizes Persian financial labels, and prepares structured outputs for CSV, JSON, or database storage.

## Project Summary

This project demonstrates a real-world Python web scraping workflow. It collects financial report announcements from CODAL, extracts income statement data from report pages, handles both embedded JSON and HTML table structures, normalizes Persian financial labels into English field names, and stores the cleaned records in a structured database-ready format.

## Key Features

- Async web scraping with Python
- Public financial announcement collection
- Income statement extraction
- JSON parsing with HTML fallback
- Persian financial label normalization
- Database-ready structured output
- CSV/JSON sample output
- Configurable environment variables

## Tech Stack

Python, aiohttp, BeautifulSoup, pandas, SQLAlchemy, SQL Server, python-dotenv, jdatetime

## What This Project Shows

This project shows my ability to build practical web scraping solutions that go beyond simple data extraction. It includes parsing, cleaning, normalization, database preparation, and output generation.
